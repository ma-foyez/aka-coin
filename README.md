# Aka.coin

MERN stack web application 

---
# How to run this application 
## Installation

1. Clone git repository

```bash
https://github.com/ma-foyez/aka-coin.git
```

2. Go to the folder

```bash
cd aka-coin
```

3. Install Node packages 

```bash
npm i
```

4. Run Local server with development change watching...

```bash
npm start
```

5. Open browser

```bash
http://localhost:3000
```
